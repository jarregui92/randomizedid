# randomizedid
You can get custom and random identifiers for your data with no length limit and other interesting features.

uses:
randimizedid() to get 10 characters random ID
returns: 'xIHJfPrCEm'


ramdomizedid(25) sends value as numeric to get random ID with that length
returns: 'LUuBXRtyO0APurPL52hoU45D7'


randomizedid(29, 5) pass a second numeric argument to get an ID separated by hyphens every second arguments lenght
returns: '71IhP-blAzp-QRhLA-qxOuB-DRKaH'
